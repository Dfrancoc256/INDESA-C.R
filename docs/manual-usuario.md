# Manual de usuario

La aplicacion tiene dos zonas:

- Sitio publico: inicio, catalogo, detalle de producto, reserva y contacto.
- Administracion: login, dashboard, productos, categorias, inventario, reservas y usuarios.

Para hacer una reserva, el usuario publico selecciona un producto y envia sus datos de contacto. En administracion, un usuario autorizado puede revisar la reserva y cambiar su estado.
